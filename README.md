# CareerAI Assistant

CareerAI Assistant is an AI powered web application that helps users improve their career preparation.

## Features
- AI Cover Letter Generator
- Career Quiz Generator
- AI Career Guidance
- User Authentication
- Personalized Career Suggestions

## Tech Stack
- Next.js
- Prisma
- PostgreSQL
- Tailwind CSS
- Gemini AI

## Setup

1. Clone the repository
2. Install dependencies

npm install

3. Create a ".env" file

DATABASE_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
GEMINI_API_KEY=

4. Run the project

npm run dev
