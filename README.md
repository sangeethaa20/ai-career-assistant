# CareerAI Assistant

CareerAI Assistant is an AI powered web application that helps users improve their career preparation.

## Features
- AI Cover Letter Generator
- Career Quiz Generator
- AI Career Guidance
- User Authentication
- Personalized Career Suggestions

## Tech Stack
- Next.js
- Prisma
- PostgreSQL
- Tailwind CSS
- Gemini AI

## Setup

1. Install dependencies

npm install

2. Create a ".env" file

DATABASE_URL=

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/onboarding
NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/onboarding

GEMINI_API_KEY=

3. Run the project

npm run dev
